# 🏥 Hospital Emergency Room Dashboard | Excel

## Project Overview
This project presents an interactive Hospital Emergency Room Dashboard built in Microsoft Excel to analyze patient flow, waiting times, satisfaction scores, admissions, demographics, and departmental referrals. The dashboard transforms raw healthcare data into meaningful insights through dynamic visualizations and interactive filters.

## Key Features
- Patient Volume Analysis
- Average Wait Time Tracking
- Patient Satisfaction Monitoring
- Admission vs Non-Admission Analysis
- Age Group and Gender Distribution
- Department Referral Insights
- Interactive Slicers and Filters

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Business Value
- Monitor emergency room performance
- Identify patient flow trends
- Improve operational efficiency
- Support data-driven healthcare decisions
